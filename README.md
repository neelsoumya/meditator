# meditator

## Introduction

Miscellaneous tools for meditation


## Installation

```r
install.packages(devtools)
library(devtools)
devtools::install_github("jacobkap/meditations")
devtools::install_github("jfisher-usgs/meditate")
```

## Usage

```r
library(meditations)
library(meditate)

meditations::meditations()

meditate::Meditate()

```

or

```r

R --no-save < meditator.R

```
